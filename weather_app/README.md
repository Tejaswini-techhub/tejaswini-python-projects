Weather App
-----------
1. Sign up at https://openweathermap.org and get a free API key.
2. Set environment variable (Linux/Mac): export OPENWEATHER_API_KEY='YOURKEY'
   On Windows (PowerShell): $env:OPENWEATHER_API_KEY='YOURKEY'
3. Install requests: pip install requests
4. Run: python3 weather_app.py
